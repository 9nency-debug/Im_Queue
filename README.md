# Im_Queue

# 🚀 IMQ — Intensive Mastery: Queue

> A focused and disciplined journey to truly understand Queue & BFS.

IMQ is a structured collection of **50 carefully curated problems** designed to build strong fundamentals and deeper intuition in:

🧱 Queue & Deque fundamentals  
👑 Monotonic Queue (sliding window techniques)  
🌐 BFS & multi-source BFS  
⏱️ Scheduling with Priority Queues  
🧠 Heap + Deque optimization  
🗺️ State-based graph traversal  

Problems are arranged in **progressive difficulty**, encouraging step-by-step learning instead of random grinding.

This repository is meant for students who want to:
- Think clearly about data structures  
- Recognize patterns instead of memorizing solutions  
- Write clean, efficient implementations  

IMQ is not about rushing or shortcuts.  
It is about **learning with structure, patience, and consistency**.
